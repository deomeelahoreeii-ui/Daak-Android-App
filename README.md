# Daak Android App (Obtainium Distribution)

This repository is intentionally release-only.
It does **not** contain app source code.

Use this repo with Obtainium to install and receive updates.

Repository URL:
- https://github.com/deomeelahoreeii-ui/Daak-Android-App

Package ID:
- `com.daak.mobile`

## Install With Obtainium (Step by Step)

1. Install Obtainium on your Android phone.
2. Open Obtainium and tap `Add App`.
3. In `App Source URL`, paste:
   `https://github.com/deomeelahoreeii-ui/Daak-Android-App`
4. Set source type to `GitHub` (if not auto-detected).
5. Confirm package ID is `com.daak.mobile`.
6. Enable:
   - `Track pre-releases` = OFF (recommended for stable installs)
   - `Background update checks` = ON
   - `Auto-install updates` = optional, your preference
7. Save the app entry.
8. Tap `Check for updates`, then install the latest release APK.

## How Updates Work

- New app builds are published in the repo `Releases` section.
- Obtainium compares release versions and prompts you to update.
- You can manually trigger checks anytime from Obtainium.

## Troubleshooting

- If install is blocked, allow installs from unknown apps for Obtainium.
- If update is not detected, pull-to-refresh in Obtainium and check network.
- If package mismatch appears, ensure package is `com.daak.mobile`.
